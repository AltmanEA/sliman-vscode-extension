# Changelog

Все важные изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-02-02

### Добавлено
- **Управление курсами**
  - Создание новых курсов с готовой структурой
  - Сканирование и анализ существующих курсов
  - Чтение и запись конфигурации курса (sliman.json)

- **Управление лекциями**
  - Добавление новых лекций с автогенерацией имен папок
  - Создание структуры лекций (slides.md, package.json)
  - Установка зависимостей для лекций
  - Обновление конфигурации курса при добавлении лекций

- **Course Explorer**
  - Древовидный интерфейс для навигации по курсу в боковой панели VS Code
  - Отображение структуры курса: действия + список лекций
  - Автоматическое обновление при изменениях в файловой системе
  - Быстрый доступ к командам: View, Edit, Build, Delete для каждой лекции
  - Действия курса: Add Lecture, View Course

- **Сборка и публикация**
  - Сборка отдельных лекций в статические файлы
  - Пакетная сборка всего курса
  - Запуск dev сервера для разработки
  - Локальный HTTP сервер для просмотра курса
  - Настройка GitHub Pages для автоматического деплоя

- **Команды расширения**
  - `sliman.createCourse` → "Create Course" - Создание нового курса
  - `sliman.scanCourse` → "Scan Course" - Сканирование курса
  - `sliman.addLecture` → "Add Lecture" - Добавление новой лекции
  - `sliman.editLecture` → "Edit Lecture" - Редактирование лекции с dev сервером
  - `sliman.deleteLecture` → "Delete Lecture" - Удаление лекции
  - `sliman.buildLecture` → "Build Lecture" - Сборка лекции
  - `sliman.viewCourse` → "View Course" - Просмотр курса
  - `sliman.openSlides` → "Open slides.md" - Открытие файла слайдов
  - `courseExplorer.refresh` → "Refresh" - Обновление Course Explorer

- **Утилиты и вспомогательные функции**
  - Транслитерация кириллицы в латиницу для имен папок
  - Валидация имен курсов и лекций
  - Управление контейнером менеджеров (Singleton паттерн)
  - Обработка ошибок и прогресс-индикаторы

- **Шаблоны**
  - Готовые шаблоны для лекций (slides.md)
  - Шаблон главной страницы курса (index.html)
  - Шаблон package.json для лекций
  - GitHub Actions workflow для автоматического деплоя

- **Тестирование**
  - Модульные тесты для всех основных компонентов
  - Интеграционные тесты с использованием @vscode/test-electron
  - Тестовые утилиты для работы с временными директориями
  - Категорийные тесты для разных менеджеров

### Архитектура
- **Модульная структура**: Разделение ответственности между менеджерами
- **TypeScript**: Полная типизация с строгими настройками
- **ESLint**: Современная конфигурация линтинга
- **VS Code API**: Интеграция с официальным API VS Code

### Технические детали
- **Поддерживаемые платформы**: Windows, macOS, Linux
- **Минимальная версия VS Code**: 1.85.0
- **Минимальная версия Node.js**: 18.18.0
- **Рекомендуемый пакетный менеджер**: pnpm
