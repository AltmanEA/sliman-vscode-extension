# Welcome to your VS Code Extension

## What's in the folder

- `src/extension.ts` - The main source file
- `package.json` - The manifest file
- `tsconfig.json` - TypeScript configuration
- `eslint.config.js` - ESLint configuration

## Get up and running straight away

1. Press `F5` to start debugging and open a new Extension Development Host window
2. Run the command "Hello World" from the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
3. You should see a notification message

## Make changes

1. Change code in `src/extension.ts`
2. Press `Ctrl+R` (or `Cmd+R`) in the Extension Development Host window to reload
